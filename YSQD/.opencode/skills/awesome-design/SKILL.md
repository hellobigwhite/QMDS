## How to Use

1. The user picks a brand from the list above (or you suggest one based on their needs)
2. Read the corresponding `DESIGN.md` file from `D:\YSQD\awesome-design-md\design-md\<brand>\DESIGN.md`
3. Apply its design tokens (colors, typography, spacing, component styles) when generating UI code
4. The `DESIGN.md` contains complete specifications: color palette with hex codes, font families with sizes, button/card/input styles, layout principles, and responsive breakpoints

## Location

All design files are at:
```
D:\YSQD\awesome-design-md\design-md\<brand-name>\DESIGN.md
```

Example: `D:\YSQD\awesome-design-md\design-md\stripe\DESIGN.md`
